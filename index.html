Elegant

Смотреть сайт Смотреть на GitHub

Учебный многостраничный сайт компании Elegant. На сайте используется семантическая разметка HTML5, стили написаны с помощью препроцессора SCSS. В разметке используется БЭМ-методология, верстка является валидной. Сайт адаптивный, благодаря этому все элементы и блоки сайта качественно отображаются на экранах различных устройств - компьютерах, смартфонах, планшетах.

В проекте используется библиотека jQvery. На главной странице используется готовое решение анимации с помощью API Canvas. Так же применяется библиотеки анимации Animate.css и WOW.js. Для демонстрации работы кнопок используется плагин sweetalert2.

Для автоматизации работы используется таск-менеджер Gulp.

Boutique

Смотреть сайт Смотреть на GitHub

Тестовое задание работодателя. Требовалось сделать декстопную и мобильную версию по макету. На сайте используется семантическая разметка HTML5, стили написаны с помощью препроцессора SCSS. В разметке используется БЭМ-методология, верстка является валидной. Сайт является адаптивным, благодаря этому все элементы и блоки сайта качественно отображаются на экранах различных устройств - компьютерах, смартфонах, планшетах.

Основные требования задания: видеофон на декстопной версии, при скролле видео плавно исчезает, меню в шапке сайта появляется при наведении, валидная форма с кастомным скроллом и чекбоксом. Так же при скролле блок с товарами едет вверх. В мобильной версии присутсвует многоуровненый аккордион.

Работа выполнена с помощью jQvery. Для валидации формы используется jQuery Validation Plugin и готовая маска для ввода номера в форму. Для автоматизации работы используется таск-менеджер Gulp.
